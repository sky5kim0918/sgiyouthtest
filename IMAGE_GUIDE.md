# 이미지 제작 가이드

## 📐 이미지 사이즈

### 1. 문항 배경 이미지 (Test 페이지)

**경로**: `public/questions/`  
**파일명**: `q1.jpg`, `q2.jpg`, ..., `q12.jpg`  
**권장 사이즈**: **1080 x 1920 픽셀** (9:16 비율)

**이유**:
- 모바일 세로 화면에 최적화된 비율
- 대부분의 스마트폰 화면 비율과 일치
- `object-cover`로 화면 전체를 채움

**제작 시 주의사항**:
- 중요한 내용은 화면 중앙과 상단에 배치
- 하단 20-30% 영역은 선택지 텍스트 박스가 오버레이됨
- 좌/우 절반 영역 모두 터치 가능하므로, 중요한 요소는 중앙에 배치

---

### 2. 결과 이미지 (Result 페이지)

**경로**: `public/results/`  
**파일명**: `A.png`, `B.png`, `C.png`, `D.png`, `E.png`  
**권장 사이즈**: **1080 x 1920 픽셀** (9:16 비율)

**이유**:
- Test 페이지와 동일한 비율로 일관성 유지
- `aspect-[9/16]`로 표시됨
- 워터마크가 오른쪽 하단에 추가됨

**제작 시 주의사항**:
- 오른쪽 하단 약 150x50 픽셀 영역은 워터마크(이름)가 표시됨
- 중요한 내용은 해당 영역을 피해서 배치

---

## 🎯 터치 영역 설명

### Test 페이지 터치 영역

```
┌─────────────────────────────────┐
│         상단 진행 표시             │
│    [이전]        [3/12]          │
├─────────────────────────────────┤
│                                 │
│                                 │
│         배경 이미지                │
│                                 │
│                                 │
│                                 │
├──────────────┬──────────────────┤
│              │                  │
│   좌측 영역    │    우측 영역        │
│  (50% 화면)   │   (50% 화면)      │
│              │                  │
│  [좌측 선택지]  │  [우측 선택지]      │
└──────────────┴──────────────────┘
```

**터치 영역 구조**:
- **좌측 영역**: 화면 왼쪽 절반 (0% ~ 50%)
- **우측 영역**: 화면 오른쪽 절반 (50% ~ 100%)
- 각 영역은 **화면 전체 높이**에 걸쳐 터치 가능
- 선택지 텍스트는 하단에 반투명 박스로 표시됨

**코드 구조**:
```tsx
<div className="absolute inset-0 z-10 flex">
  {/* 좌측 버튼 - 화면의 50% */}
  <button className="flex-1">...</button>
  
  {/* 우측 버튼 - 화면의 50% */}
  <button className="flex-1">...</button>
</div>
```

## 📝 이미지 제작 팁

### 1. 안전 영역 (Safe Zone)

**상단**: 
- 상단 80px는 진행 표시와 이전 버튼이 표시됨
- 중요한 내용은 그 아래에 배치

**하단**:
- 하단 약 150px는 선택지 텍스트 박스가 표시됨
- 중요한 내용은 그 위에 배치

**좌/우 중앙**:
- 화면을 좌/우로 나누는 중앙선(50% 지점) 근처는 터치 영역이 겹칠 수 있음
- 중요한 요소는 중앙에서 약간 벗어나게 배치 권장

### 2. 텍스트 가독성

- 배경 이미지 위에 텍스트를 넣을 경우, 가독성을 위해 그림자나 배경 박스 사용 권장
- 선택지 텍스트는 자동으로 반투명 박스(`bg-black/60`) 위에 표시됨

### 3. 이미지 포맷

- **문항 배경**: JPG 권장 (용량 최적화)
- **결과 이미지**: PNG 권장 (투명도 필요 시, 워터마크 추가 시)

### 4. 파일 크기

- 모바일 환경을 고려하여 각 이미지당 **500KB 이하** 권장
- 너무 크면 로딩이 느려질 수 있음

## ✅ 체크리스트

이미지 제작 전 확인사항:

- [ ] 문항 배경 이미지: 1080x1920 사이즈 (9:16 비율)
- [ ] 결과 이미지: 1080x1920 사이즈 (9:16 비율)
- [ ] 파일명이 정확한지 확인 (`q1.jpg` ~ `q12.jpg`, `A.png` ~ `E.png`)
- [ ] 파일이 올바른 폴더에 있는지 확인 (`public/questions/`, `public/results/`)
- [ ] 이미지 파일 크기가 500KB 이하인지 확인
- [ ] 중요한 콘텐츠가 안전 영역(상단 80px, 하단 150px) 밖에 있는지 확인
- [ ] 좌/우 중앙선(50% 지점)에 중요한 요소가 없는지 확인
